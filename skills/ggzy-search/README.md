# ggzy-search

全国公共资源交易平台（ggzy.gov.cn）公告搜索与报告生成 Skill，适用于 [OpenClaw](https://openclaw.app) 平台。

## 功能简介

自动在全国公共资源交易平台上按条件搜索交易公告，支持**单关键词或多关键词**批量搜索，翻页抓取结果，并生成按关键词分组的结构化 HTML 报告。适用于招投标信息监控、市场调研、竞品分析等场景。

## 默认筛选条件

| 条件 | 默认值 | 是否可覆盖 |
|------|--------|-----------|
| 发布时间 | 近十天 | 暂不支持 |
| 省份 | 广东 | 可覆盖 |
| 信息类型 | 交易公告 | 暂不支持 |
| 关键词 | 无（必填，支持多个） | 用户指定 |
| 翻页数 | 全部 | 可指定页数（对每个关键词生效） |

## 使用方式

### 基本用法

直接用自然语言告诉 OpenClaw 你要搜索什么，Skill 会自动触发：

```
帮我搜索广东近十天的医疗设备招标公告
```

```
查一下最近广东有没有足球相关的交易公告
```

```
搜索广东近十天关于教育信息化的招标信息
```

### 指定翻页数

```
帮我搜索广东的医疗招标公告，翻3页，每页截图
```

### 多关键词搜索

```
帮我搜索广东近十天关于医疗和足球的招标公告
```

```
搜索以下关键词的广东招标公告：医疗、足球、教育
```

多关键词会逐个搜索，最终汇总到一份按关键词分组的 HTML 报告中。

### 指定省份

```
帮我搜索浙江近十天的建筑工程招标公告
```

### 触发关键词

以下关键词出现在对话中时会自动触发此 Skill：

> 招标、投标、中标、招标公告、中标公告、采购公告、政府采购、公共资源交易、交易公告、工程招标、货物招标、服务招标、竞争性谈判、竞争性磋商、询价公告、单一来源、资格预审、建设工程、土地使用权、矿业权、国有产权

## 输出内容

1. **HTML 报告文件**：一份报告包含所有关键词的结果，按关键词分组展示（搜索条件摘要、每组记录数、公告明细表格、原文链接）
2. **每页截图**：搜索结果每一页的浏览器截图

## 前置条件

- OpenClaw 浏览器服务可用（Browser Relay 扩展已开启，或 Sandbox 浏览器已启用）
- 网络可访问 `https://www.ggzy.gov.cn`

## 工作原理

此网站为纯 JavaScript 动态渲染页面，**不支持 URL 查询参数**。Skill 通过以下方式操作：

1. 从用户输入中解析关键词列表（支持自然语言和列举式）
2. 导航到基础 URL（不带任何参数）
3. 对每个关键词循环执行：设置筛选条件 → 搜索 → 验证 → 抓取 → 翻页
4. 所有关键词搜完后，汇总无结果的词批量询问用户是否重搜
5. 生成按关键词分组的 HTML 报告

## 常见问题

**Q：搜索结果没有按省份过滤？**
A：省份下拉框的 `option` 文本必须精确匹配（如"广东"而非"广东省"），如果不生效会自动重试。

**Q：结果页面显示"页面正在努力加载中"？**
A：网站内容通过 AJAX 异步加载，需要等待 2-3 秒。Skill 已内置等待逻辑。

**Q：搜索不到结果怎么办？**
A：如果是多关键词搜索，Skill 会先跑完所有关键词，然后把无结果的词汇总起来一次性告知你。对于多词组合（如"新冠医疗"），Skill 会识别出主体词（如"新冠"）并建议重搜。你可以逐个决定哪些要用主体词重搜、哪些换词、哪些跳过。重搜仍无结果会再次询问，直到你决定结束。

**Q：多关键词搜索会生成几份报告？**
A：只生成一份 HTML 报告，按关键词分组展示。每个关键词是报告中的一个独立章节。

**Q：浏览器服务连不上？**
A：确认 OpenClaw Browser Relay 扩展已开启（Chrome 工具栏图标显示 ON），或在 OpenClaw 设置中启用 Sandbox 浏览器。
